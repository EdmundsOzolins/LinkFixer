Change XWiki with curl

Install:

	sudo apt install curl

Use:

	curl -u user:password -X PUT -T "@content" -H "Content-Type: text/plain" 
       		"http://localhost:8080/xwiki/rest/wikis/xwiki/spaces/Sandbox/pages/{PageName}"

Example:

curl -u admin:admin -X PUT -T "/home/student/Desktop/CurlInfo.txt" -H "Content-Type: text/plain" "http://localhost:8080/xwiki/rest/wikis/xwiki/spaces/TestPageLinkChecker/spaces/SimplePage/pages/WebHome"
